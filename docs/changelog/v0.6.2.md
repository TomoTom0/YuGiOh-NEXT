# v0.6.2 (2025-12-22)

## New Features

- console.temp() エイリアスを追加。一時的なデバッグログ用でconsole.debugへのエイリアス。利用後は必ず削除すること。（TASK-242）
- オプションページに詳細なキャッシュ削除機能を追加。カードDB、拡張機能設定、デッキ情報、デッキサムネイル、その他を個別選択可能。（TASK-240）

## Bug Fixes

- デッキが空の場合にサムネイル生成で不要な警告が出力されていた問題を修正。デッキが空の場合は正常な動作として警告を出さないように変更。（TASK-244）
- デッキ編集画面で同じカードを自分自身にドロップした場合に無意味な履歴が追加される問題を修正。sourceUuid === targetUuid の場合に早期リターンを追加し、カードが実際に移動しない場合は履歴に追加しないように修正。（TASK-246）

## Changes

- オプションページのソート順選択肢を簡潔に整理。「リリース日降順」→「新しい順」、「リリース日昇順」→「古い順」、「名前昇順（あいうえお順）」→「あいうえお順」に変更。不要な選択肢（「公式順」「名前降順」）を削除。（TASK-248）
- ソート順選択肢をカード名順とリリース日順の2つのみに簡素化。最もシンプルでわかりやすい形に整理。（TASK-248）
- オプションページの通知メッセージのデザインを改善。青い情報メッセージ（info）と赤いエラーメッセージ（error）に分離し、フォントウェイト、角丸、配色を改善。（TASK-234）
- オプションページのデッキ表示画面のGeneralセクションを改善。大見出しを追加、カード画像サイズのボタンレイアウトを横並びから折り返し可能に変更、説明文を簡潔に修正。（TASK-243）

## Performance

- デッキ保存時の体感待ち時間を削除。ytknプリロード機能を追加し（編集ページ表示時に事前取得、保存成功後に次回用を非同期プリロード）、保存ボタンクリック後の遅延時間（2秒→0秒）を削減。（TASK-245）
