# v0.5.8 (2025-12-13)

## New Features

- 検索入力欄で履歴やお気に入りから過去の検索条件を選択できる `/h` と `/f` のスラッシュコマンドを追加 (#128)

## Bug Fixes

- 検索UI関連のフォントサイズを専用CSS変数(--search-ui-font-size)で管理し、Right Areaの設定に影響されないように修正。検索条件チップのサイズも縮小して、はみ出る問題を解決 (#127, #131, #141)
- 検索入力欄上部の検索条件クリアボタンが機能しなかった問題を修正。searchStore.clearAllFilters()メソッドを追加 (#136)
- デッキ編集画面の初期表示時にカテゴリ優先アイコンが表示されない問題を修正。UnifiedCacheDBの初期化タイミングを改善し、categoryMatchedCardIdsのカード情報取得をunifiedDB.reconstructCardInfo()に変更 (#140)

## UI Improvements

- カードにホバーした際のボタンが優先配置矢印アイコンより手前に表示されるようにz-indexを調整 (#126)
- Right Areaのデフォルト幅とフォントサイズをLに変更（より読みやすく） (#129)
- デッキ名入力欄の最大幅を600pxに拡大、画面幅に余裕がある場合はより広く表示 (#133)
- 検索入力バーに薄いborderを追加（var(--border-primary)）。右上以外の配置でも境界が明確になり視認性が向上。また、ハードコードされた色値をCSS変数に置き換え (#134)
- 検索条件ダイアログの「検索条件」タブにフィルター条件数のバッジを追加 (#135)
- カード画像のhover時に表示されるボタンアイコンのサイズを、画像サイズ設定に応じて自動調整するように修正（カード幅の17%） (#138)

## Refactoring

- filterChips配列を削除。常に空配列だったため、全ての参照箇所から削除し、コードを簡素化。useSearchFilters、useFilterInput、SearchInputBar.vue、SearchFilterChips.vueから不要なコードを削除 (#139)

## Documentation

- READMEの画像パスをwebp形式に更新、存在しない画像への参照を削除 (#132)
